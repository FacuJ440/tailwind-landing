Uso Práctico

Tailwind CSS puede utilizarse de varias maneras según el tamaño y las necesidades del proyecto. Su flexibilidad permite integrarlo tanto en proyectos pequeños como en aplicaciones web complejas.

1. Instalación rápida mediante CDN

Ideal para demostraciones o proyectos pequeños que no requieren personalización avanzada. Solo se necesita incluir el siguiente enlace dentro del head del archivo HTML:

<link href="https://cdn.jsdelivr.net/npm/tailwindcss@3.4.0/dist/tailwind.min.css" rel="stylesheet">

Este método permite comenzar a usar Tailwind de inmediato, aunque no permite personalización del archivo tailwind.config.js.

2. Instalación mediante npm (recomendada)

Ideal para proyectos profesionales. Permite personalizar completamente el diseño, optimizar el CSS y usar herramientas modernas.

npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
content: ["./index.html", "./src/**/*.{js,ts,jsx,tsx}"]
@tailwind base;
@tailwind components;
@tailwind utilities;

3. Instalación en frameworks populares

  • React (Vite o CRA): instalación igual que con npm, agregando el CSS principal en index.css.
  • Next.js: Tailwind se configura automáticamente con postcss.config.js.
  • Vue/Nuxt: agregar las directivas @tailwind en el CSS global.
  • Angular: se integra mediante PostCSS en el archivo angular.json.

En resumen, Tailwind CSS ofrece distintas formas de implementación: desde la inclusión rápida por CDN hasta configuraciones profesionales con npm y frameworks modernos, garantizando flexibilidad, eficiencia y escalabilidad.
